# Sistema DST - Gestión de Reparaciones

Aplicación de escritorio para la gestión de un taller de reparación de equipos informáticos: clientes, equipos y órdenes de reparación, con base de datos local SQLite.

## Tecnologías
- Python 3.x
- PySide6 (interfaz gráfica)
- qfluentwidgets (componentes visuales)
- SQLite (base de datos local)

## Cómo correrlo desde código fuente
1. Cloná el repositorio
2. Instalá las dependencias: `pip install -r requirements.txt`
3. Ejecutá: `python app.py`

## Descarga directa
Si preferís no instalar Python, descargá el ejecutable ya compilado desde la sección [Releases](enlace-a-tu-release) de este repositorio.

> ⚠️ Windows puede mostrar una advertencia de SmartScreen al abrir el .exe por primera vez, ya que no está firmado digitalmente. Es esperable en proyectos personales — podés confirmar la ejecución desde "Más información" → "Ejecutar de todas formas".
